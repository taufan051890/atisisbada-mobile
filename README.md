# Atisisbada-mobile


### Installation
1. Install Ionic 
   ```npm install -g @ionic/cli```
2. Clone the repo
   ```sh
   git clone https://github.com/taufan051890/atisisbada-mobile.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run Aplication ATISIS-PANDA 
   ```
    cd myApp
    ionic serve
    ```
